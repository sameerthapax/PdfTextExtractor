# Qbee PDF Extractor Backend

This is a lightweight Node.js backend for extracting text from uploaded PDF files using `pdf-parse`.

## Endpoints

- `POST https://pdftextextractor-wl9d.onrender.com/extract`: Upload a PDF (form-data key = `file`) and get extracted text

## Usage

1. Run `npm install`
2. Run `node server.js`
